#ssmeasyui demo
## 技术栈
* spring+springmvc+mybatis
* poi3
* 自定义标签开发
* easyui1.4.4
* druid
* weui
* log4jdbc
* hibernate-validator

##How to run
项目采用maven构建，运行前请先执行src/main/java下的sql脚本到你的mysql数据库中，然后修改src/main/resource下的jdbc.properties配置文件中的数据库配置信息，进入项目目录，执行mvn tomcat7:run来运行
